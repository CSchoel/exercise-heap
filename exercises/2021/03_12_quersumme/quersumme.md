# Quersumme

Schreiben Sie eine eine Funktion `digit_sum(x)` in der Datei `digit_sum.py`, die die Quersumme der Zahl `x` bestimmt.

Beispiel:

* `digit_sum(153)` = `1 + 5 + 3 = 9`

> Tipp: Sie können die Funktion sich auch selbst rekursiv aufrufen lassen.
